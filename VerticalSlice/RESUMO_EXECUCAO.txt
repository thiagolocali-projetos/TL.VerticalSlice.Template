🚀 COMO TESTAR
═══════════════════════════════════════════════════════════════════════════

1. Executar API:
   cd src
   dotnet run --project TL.Exemplo.API

2. Abrir Swagger:
   https://localhost:5001

3. Fazer Login:
   POST /api/v1/auth/login
   {"username":"admin","password":"admin123"}

4. Copiar Token e usar no "Authorize"

5. Testar endpoints:
   - GET /api/v1/produtos
   - GET /api/v1/produtos/paged
   - POST /api/v1/produtos
   - PUT /api/v1/produtos/{id}
   - DELETE /api/v1/produtos/{id}

6. Testar Health:
   GET /health
   GET /health/ready
   GET /health/live

7. Executar Testes:
   dotnet test
